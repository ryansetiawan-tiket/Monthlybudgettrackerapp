## Responsive Breakpoints

### Mobile (< 768px)
- Buttons stack vertically in header
- Checkbox touch target: 44px minimum
- Dialog takes 90% width
- Item list uses full width

### Tablet (768px - 1024px)
- Header buttons horizontal with wrapping
- Standard checkbox size
- Dialog max-width: 600px
- 2-column layout possible for small items

### Desktop (> 1024px)
- All inline, no wrapping
- Standard sizing
- Dialog max-width: 700px
- Optimal spacing
